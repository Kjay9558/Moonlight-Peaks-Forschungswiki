# Arbeitsablauf

## Archivimport

1. Originalquelle unverändert unter `archive/` sichern.
2. Nachrichten oder Medien mit stabiler Quellenreferenz erfassen.
3. Einzelne bestätigte Aussagen als Beobachtungen eintragen.
4. Entitäten aktualisieren.
5. Wiki-Seiten und Changelog aktualisieren.
6. Versionsnummer erhöhen und Git-Commit erstellen.

## Regel für neue Chatdaten

Jeder neue Fund wird zuerst als Beobachtung gespeichert. Erst danach wird der zusammengeführte Datensatz geändert. So bleiben Herkunft, Korrekturen und Unsicherheiten nachvollziehbar.

## Git-Commit-Konvention

- `data: Seelentropfen ergänzen`
- `fix: Farbe von George Flair korrigieren`
- `docs: Bienenstock-Mechanik beschreiben`
- `import: Chatturns 1–50 archivieren`
